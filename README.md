如何运行此项目：

python==3.9
```bash
git clone https://github.com/populuscathayana/STRUCTURE_outside.git
cd STR
conda create -n STR python==3.9
conda activate STR
pip install -r requirements.txt
sudo apt-get update
sudo apt-get install libxrender1
streamlit run app.py
```
